# koishi-plugin-slave-market

[![npm](https://img.shields.io/npm/v/koishi-plugin-slave-market?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-slave-market)

[![爱发电](https://img.shields.io/badge/爱发电-支持作者-FF4D4D?style=flat-square)](https://afdian.com/item/fd7899e40b2711f0a67d52540025c377)

大奴隶时代 - 一个基于 Koishi 的群聊游戏插件

## 功能特点

### 基础系统
- 🏦 银行系统：存款、取款、利息、转账
- 👥 奴隶市场：购买、训练、决斗
- 🌾 种地系统：种植、收获、作物状态
- 🛡️ 保镖系统：雇佣保镖、保护状态
- ⚔️ 反抗系统：奴隶反抗、获得自由
- 🏪 道具商城：购买、使用、管理外观
- 🎮 决斗系统：奴隶之间的战斗
- 🎲 训练系统：提升奴隶属性

### 游戏特色
- 💰 经济系统：完整的货币流通体系
- 📈 成长系统：身价、信用等级、存款上限
- ⏰ 冷却系统：合理的操作冷却时间
- 🎯 概率系统：训练、决斗、反抗等概率机制
- 🎮 互动系统：玩家之间的交易与互动
- 🏆 成就系统：完成特定目标获得奖励

## 快速开始

1. 安装插件
   ```bash
   npm install koishi-plugin-slave-market
   ```
2. 在 Koishi 配置文件中启用插件
   ```js
   export default {
     plugins: {
       'slave-market': {
         // 参考下文“配置说明”，根据需要调整
       },
     },
   }
   ```
3. 发送 `注册玩家 <昵称> <性别>` 完成注册，输入 `我的信息` 查看状态或 `玩家帮助` 获取内置帮助图像。

## 使用说明

### 基础指令
- `注册玩家 [昵称] [性别]` - 注册成为玩家
- `我的信息` - 查看个人信息
- `奴隶市场` - 查看可购买的玩家
- `我的奴隶` - 查看拥有的奴隶
- `排行榜` - 查看游戏排行榜

### 银行指令
- `存款 [金额]` - 存入银行
- `取款 [金额]` - 从银行取出
- `领取利息` - 领取存款利息
- `银行信息` - 查看银行账户信息
- `升级信用` - 提升信用等级
- `转账 @用户 [金额]` - 向指定用户转账

### 奴隶指令
- `购买玩家 @用户` - 购买指定用户
- `赎身` - 支付赎金获得自由
- `放生 @用户` - 解除与指定奴隶的关系
- `训练 @用户` - 训练指定奴隶
- `决斗 @奴隶1 @奴隶2` - 让两个奴隶决斗
- `奴隶信息 @用户` - 查看指定奴隶的详细信息

### 种地指令
- `种地 [作物名称]` - 种植指定作物
- `收获` - 收获已成熟的作物
- `作物状态` - 查看当前作物状态
- `作物列表` - 查看可种植的作物

### 保镖指令
- `保镖市场` - 查看可雇佣的保镖
- `雇佣保镖 [保镖名称]` - 雇佣指定保镖
- `保镖状态` - 查看当前保镖状态
- `解雇保镖` - 解雇当前保镖

### 反抗指令
- `反抗` - 发动反抗获得自由
- `反抗状态` - 查看反抗状态
- `反抗记录` - 查看反抗历史

### 其他指令
- `打工` - 打工赚钱
- `抢劫 @用户` - 抢劫指定用户
- `修炼` - 消耗金币提升身价
- `商城` - 查看道具商城
- `购买道具 [道具名称]` - 购买指定道具
- `道具背包` - 查看背包
- `使用道具 [道具名称]` - 使用指定道具
- `每日签到` - 领取每日奖励

## 游戏规则

### 初始设定
- 初始身价：100
- 初始存款上限：1000
- 初始信用等级：1
- 初始余额：0
- 初始反抗成功率：30%
- 初始训练成功率：50%

### 收入来源
- 打工：基础收入为身价的10%
- 抢劫：可抢劫目标30%的余额
- 存款利息：每小时1%的利息收入
- 每日签到：随机奖励
- 奴隶劳动：奴隶可产生收入

### 特殊机制
- 金融危机：系统余额低于5万时触发
- 游戏胜利：拥有100个以上奴隶且每个身价超过10万
- 游戏失败：系统余额小于0
- 反抗系统：奴隶可以通过反抗获得自由
- 训练系统：提升奴隶属性
- 决斗系统：奴隶之间可以决斗

### 冷却时间
- 购买：5分钟
- 打工：2分钟
- 抢劫：1分钟
- 转账：2分钟
- 反抗：1小时
- 训练：30分钟
- 决斗：15分钟
- 签到：24小时

## 注意事项
- 存款利息最多累计24小时
- 所有交易将收取一定比例手续费
- 信用等级提升可增加存款上限
- 训练和决斗都有成功/失败概率
- 反抗失败会降低反抗成功率
- 每日反抗次数有限
- 保镖可以保护玩家免受抢劫
- 奴隶可以通过训练提升属性
